C. KNOWN ISSUES — PUBLICATION NOTES
------------------------------------------------------------------------------
  • First-order HDMR limitation
        Additive decomposition f₀ + Σ gᵢ(xᵢ) cannot represent cross-variable
        interactions. Affected: Six-Hump Camel, Rosenbrock, Branin, Griewank.
        Mitigation: second-order HDMR with interaction terms (future work).

  • Branin domain asymmetry
        function_ranges.json uses [-5, 15] symmetric.  Standard domain is
        x₁ ∈ [-5, 10], x₂ ∈ [0, 15].  All 3 global minima lie within
        [-5, 15]² so results are valid, but search space is 2× larger.

  • Six-Hump Camel — f(0,0) = 0 ≠ f*
        f* = −1.0316.  Any report claiming f ≈ 0 as optimal is incorrect;
        (0, 0) is a saddle point.  v1 compiler reported this as 'success'.

  • Rosenbrock — f(0,0) = 1.0 ≠ f*
        f* = 0.  Optimizer stuck at origin due to narrow curved valley
        that first-order HDMR cannot resolve.

  • Previous Improvement % bug
        v1 used abs(f_found) in denominator, completely masking negative
        optima.  v2 uses signed gap-to-optimum throughout.
